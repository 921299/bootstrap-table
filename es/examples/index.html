Basic

Table from html

Transforma una tabla html, sin formato. by @wenzhixin

Table from data

Crea una tabla desde datos existentes. by @wenzhixin

Table from url

Crea una tabla desde una url. by @wenzhixin

Style

Classes

Use las opciones height, classes, striped y las opciones de columna class, width para setear los estilos de bootstrap table. by @wenzhixin

RowStyle

Use la opción rowStyle para setear el estilo de la fila de bootstrap table. by @wenzhixin

CellStyle

Use la opción de columna cellStyle para setear el estilo de la celda de bootstrap table. by @wenzhixin

Align

Use las opciones align, halign y valign para setear el alineamiento de las columnas respecto a su encabezado. by @wenzhixin

Sort

Basic Sort

Use las opciones sortName, sortOrder, sortable, y las opciones de columna sortable, order para setear el ordenamiento básico de bootstrap table. by @wenzhixin

Custom Sort

Use la opción de columna sorter para definir una ordenamiento customizable of bootstrap table. by @wenzhixin

Format

Use la opción de columnas formatter para dar formato a la columna. by @wenzhixin

Show current row count using Formatter

Use la opción de columnas formatter para mostrar el índice de la fila seleccionada. by @DominikAngerer

Hide Header

Use la opción showHeader: false para ocultar el header de bootstrap table. by @wenzhixin

Columns

Basic columns

Use las opciones showColumns, minimumCountColumns, y las opciones de columna visible, switchable para mostrar el menú lisra para mostrar/ocular las columnas. by @wenzhixin

Large columns

Bootstrap table soporta columnas grandes, se mostrará automáticamente el scroll horizontal. by @wenzhixin

Card View

Use la opción cardView: true para mostrar la vista de tarjeta. by @wenzhixin

Select

Radio Select

Use las opciones clickToSelect, selectItemName y la opción de columna radio para mostra un radiobox en la tabla. by @wenzhixin

Checkbox Select

Use la opción clickToSelect y la opción de columna checkbox para mostrar un checkbox en la tabla. by @wenzhixin

Disabled Checkbox

Use las opciones checkboxHeader, checkboxEnable y la opción de columna formatter para deshabilitar el control. by @wenzhixin

Single Checkbox

Use la opción singleSelect para permitir solo un checkbox marcado a la vez en la tabla. by @wenzhixin

Toolbar

Basic Toolbar

Use las opciones search, showColumns, showRefresh, showToggle para mostrar una barra de herramientas básica. by @wenzhixin

Custom Toolbar

Use la opción toolbar para definir una barra de herramientas customizada. by @wenzhixin

Pagination

Client Side

La opción de paginación por defecto de la tabla es client. by @wenzhixin

Server Side

Use la opción sidePagination: server para definir la paginación de la tabla del lado del servidor. by @mikepenz

Aquí está el código de la paginación server-side. https://gist.github.com/mikepenz/06df1204cbb65b874cb5 Es un api rápida, justo para saber cómo se usa y listo.

Events

Basic events

Eventos de la tabla. by @wenzhixin

Column events

Use las opciones de columna formatter, events para definir los eventos customizables. by @wenzhixin

Methods

Reset View

El encabezado de la tabla no se ajusta automáticamente, se necesita llamar al método resetView. by @wenzhixin
