Basic

Table from html

Transform table from an existing, unformatted html table. by @wenzhixin

Table from data

Transform table from an existing data. by @wenzhixin

Table from url

Transform table from an url. by @wenzhixin

Style

Classes

Use height, classes, striped options and class, width column options to set the styles of bootstrap table. by @wenzhixin

RowStyle

Use rowStyle option to set the row style of bootstrap table. by @wenzhixin

CellStyle

Use cellStyle column option to set the cell style of bootstrap table. by @wenzhixin

Align

Use align, halign and valign options to set the alignment of the columns and their header. by @wenzhixin

Sort

Basic Sort

Use sortName, sortOrder, sortable options, and sortable, order column options to set the basic sort of bootstrap table. by @wenzhixin

Custom Sort

Use sorter column option to define the custom sort of bootstrap table. by @wenzhixin

Format

Use formatter column option to format the display of bootstrap table column. by @wenzhixin

Show current row count using Formatter

Use formatter column option to display the index of the current row. by @DominikAngerer

Hide Header

Use showHeader: false to hide the header of bootstrap table. by @wenzhixin

Columns

Basic Columns

Use showColumns, minimumCountColumns options, and visible, switchable column options to show the columns menu list use to switch. by @wenzhixin

Large Columns

Bootstrap table support large columns, it will auto to show the horizontal scroll bar. by @wenzhixin

Card View

Use cardView: true option to show Card View. by @wenzhixin

Select

Radio Select

Use clickToSelect, selectItemName options and radio column option to show a radio select table. by @wenzhixin

Checkbox Select

Use clickToSelect option and checkbox column option to show a checkbox select table. by @wenzhixin

Disabled Checkbox

Use checkboxHeader, checkboxEnable options and formatter column option to disabled select input. by @wenzhixin

Single Checkbox

Use singleSelect option to allow checkbox selecting only one row. by @wenzhixin

Toolbar

Basic Toolbar

Use search, showColumns, showRefresh, showToggle options to show the basic toolbars. by @wenzhixin

Custom Toolbar

Use toolbar option to define the custom toolbars. by @wenzhixin

Pagination

Client Side

The default side pagination of table is client. by @wenzhixin

Server Side

Use sidePagination: server option to define the server side pagination of table. by @mikepenz

Here's the server-side code. https://gist.github.com/mikepenz/06df1204cbb65b874cb5 It's a quick and dirty api, just to showcase the usage.

Events

Basic Events

Table events. by @wenzhixin

Column Events

Use formatter, events column option to define the custom events. by @wenzhixin

Reset View
